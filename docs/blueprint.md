# **App Name**: BharatGate

## Core Features:

- Unified Link Repository: A clean, curated directory of every major Indian online service including Aadhaar, IRCTC, DigiLocker, and state portals.
- Smart Bharat Concierge: A natural language AI tool that understands complex citizen requests and points them directly to the relevant portal.
- State-Wise Categorization: Intuitive filtering systems to quickly narrow down services based on specific Indian states or territories.
- Essential Service Monitor: A real-time status tracker to verify if major government portals are online and functional.
- Personalized Dashboard: Allow users to pin frequently used links for one-click access to personal utilities.

## Style Guidelines:

- The primary color is a sophisticated Cobalt Blue (#2C64F1), representing trust and professional reliability.
- The background is a deep, desaturated midnight Navy (#0E121C) to maintain a modern, premium aesthetic.
- The accent color is a sharp Electric Amethyst (#824EF6), used sparingly for CTAs to create a high-contrast digital feel.
- A font pairing featuring 'Space Grotesk' (sans-serif) for professional, techy headlines and 'Inter' (sans-serif) for maximum legibility in lists.
- Minimalist line icons that clearly differentiate categories like finance, education, and legal services.
- An airy, grid-based layout that prioritizes scannability and prevents cognitive overload from many links.
- Smooth, low-latency hover transitions and fade-in states for a premium, agency-style interaction feel.
